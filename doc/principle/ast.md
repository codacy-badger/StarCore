# AST
~~_神奇语法树_~~  
语法树用来分析一个函数内的语句  
例如:  
````cpp
var(a,100)
while(a>0):
    echo(a)
    var(a,a.minus(1))
end
````
````
------------------------------------------
          -------------
          - sentences -
          -------------
            /         \
           |           ---
       ----------         \
       -exec var-      -------     -------
       ----------      -while-  <= -a > 0-
          /     \      -------     -------
         /       \         \ 
     -------   --------     ---\
     -var a-   -var 10-         |
     -------   --------     ---------
                            -  body -
                            ---------
                             /  |
           -----------------   /
          /                   /
     -----------        ----------   
     -exec echo-        -exec var-
     -----------        ----------
      /                 /      \
-------              -------   -------------
-var a-              -var a-   -exec a.minus
-------              -------   -------------
                                   |
                                -------
                                -var 1-
                                -------

````
